## Çalışma Linkleri Klasörleri Hakkında

* Eğitimlerde oluşturulan projeler ve ödevler kategorize edilerek bu repo (eğitim reposu) içerisinde klasörlenmiştir.

* İlgili çalışmalara ait klasörlerin eğitmen/denetmenlerce rahatça indirilerek incelenebilmesi için kendine ait depolama/bulut linkleri klasörleri altındaki README klasörleri altında bulabilirsiniz.
